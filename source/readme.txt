3Dフードプリンター制御用プログラムソースコード
Version 1. 0. 0  (2025/1/9)
==========================================================================
◆内容
別添の3Dフードプリンター制御プログラム<bpe3dfp.exe>および<manual.exe>のCソースコードです。

◆ビルド方法 (bpe3dfp.exe)
開発環境にMicrosoft Visual Studio 2022を使用した場合を例にビルド方法を説明します。
 ・「新しいプロジェクトの作成(N)」を選択
 ・「空のプロジェクト」を選択し、「次へ」ボタンを押す
 ・「プロジェクト名」、「場所」、「ソリューション名」を適宜設定し「作成」ボタンを押す
 ・作成したプロジェクトのフォルダにソースコードファイルをすべてコピーする
   main.cpp
   global.cpp
   parse_block.cpp
   motion_control.cpp
   device_Y2C_PC104.cpp
   global.h
   struct.h
   g_code.h
   device_Y2C_PC104.h
   motion_control.h
   parse_block.h
   configuration.h

 ・Y2Corporation (https://www.y2c.co.jp/) よりソフトウェアパッケージを入手し、以下のファイルをプロジェクトフォルダにコピーする。
   Ydu.lib
   YduApi.h
   YduPmcSApi.h
   YduResult.h

 ・Visual Studio 2022のソリューションエクスプローラより、「追加」-> 「既存の項目」を選択し、cppおよびhファイルをそれぞれプロジェクトに追加する。
 ・メニュー「プロジェクト(P)」あるいはソリューションエクスプローラより、プロジェクトのプロパティを開く。
 ・「構成」->「リンカー」->「入力」を開き、「追加の依存ファイル」を選択し、「編集」を選択する。
 ・テキストボックス「追加の依存ファイル」に、<Ydu.lib>を追記し、「OK」ボタンを押す。
   (Debug mode、Release modeを切り替えるときは、それぞれのプロパティで、<Ydu.lib>を追加する）

 ・メニュー「ビルド(B)」-> 「ソリューションのビルド」もしくは「<プロジェクト名>のビルド」より実行ファイルをビルド

◆ビルド方法 (manual.exe)
<main.cpp>ファイル以外はすべて<bpe3dfp.exe>のソースファイルと共通です。
前述の<bpe3dfp.exe>と同じ方法でビルドしてください。

◆装置定数の変更
装置定数はすべて<configuration.h>に記載しています。適宜修正してビルドしてください。
